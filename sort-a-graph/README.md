# How to sort a Graph

YouTube: [link](https://www.youtube.com/watch?v=EJoKJiod0KQ)
Whiteboard: [link](https://miro.com/app/board/uXjVGcpqFXU=)

Naive solution runs in $O(N^2)$, meaning you shouldn't use it. Optimized solution is essentially the same algorithm, but it runs in $O(N)$. Final solution is the simplest and runs using only a basic dfs. It is the recommended way to write Topological Sort
